# ChatSphere 💬

A modern real-time chat application built using:

- React
- Firebase
- Tailwind CSS
- Framer Motion
- Vite

## Features

- Real-time messaging
- Online/offline status
- Typing indicator
- Emoji picker
- Responsive mobile UI
- Glassmorphism design
- Protected routes
- Firebase authentication
